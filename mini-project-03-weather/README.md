# 🌤️ Mini Project 03: Weather App

A lightweight, responsive web application that fetches and displays real-time weather information using modern JavaScript.

## 🚀 Features
* **Live Weather Data**: Fetches current temperature, humidity, and wind speed.
* **City Search**: Dynamically updates data based on user input.

## 🛠️ Technologies Used
* **API**: [OpenWeatherMap API](https://openweathermap.org)
